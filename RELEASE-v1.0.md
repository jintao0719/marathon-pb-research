# 🎉 马拉松PB研究院 v1.0 正式发布

**发布日期**: 2026年2月27日  
**版本号**: v1.0.0  
**代号**: First Step

---

## 📋 发布概述

马拉松PB研究院是一个专业的马拉松赛事信息平台，旨在帮助跑者获取最新赛事信息、学习科学训练方法、突破个人最佳成绩（PB）。

### 🎯 核心定位
- **专业** - 提供权威的赛事信息和训练知识
- **全面** - 覆盖入门到进阶的全方位内容
- **及时** - 每日自动更新赛事数据
- **互动** - 支持用户反馈和社区交流

---

## ✨ 主要功能

### 1. 🏃 赛事专区
- ✅ 收录全国48场马拉松赛事
- ✅ 智能状态管理（报名中/即将开始/已结束）
- ✅ 多维度筛选（省份/级别/类型）
- ✅ 详细的赛事信息和路线图
- ✅ 报名链接有效性检查

### 2. 📚 入门专区
- ✅ 马拉松基础知识
- ✅ 装备选购指南
- ✅ 新手训练计划
- ✅ 常见问题解答

### 3. 🚀 进阶专区
- ✅ 专业训练方法
- ✅ 配速策略指导
- ✅ 营养补给方案
- ✅ 比赛日攻略

### 4. 💬 跑者心声
- ✅ 用户反馈提交（支持图片）
- ✅ 无需登录即可反馈
- ✅ 管理员后台管理

### 5. 🔐 管理员后台
- ✅ 赛事数据管理
- ✅ 用户反馈管理
- ✅ 数据自动更新

## 🛠️ 技术架构

### 前端
- **框架**: Vue 3 + Composition API
- **路由**: Vue Router 4
- **构建**: Vite 5
- **样式**: CSS Variables + Scoped CSS

### 后端
- **框架**: Express.js
- **定时任务**: node-cron
- **数据抓取**: Axios + Cheerio
- **数据存储**: JSON文件

### 部署
- **前端**: 静态文件（可部署到任何静态托管服务）
- **后端**: Node.js服务（支持PM2进程管理）

## 🚀 快速开始

### 本地开发

```bash
# 克隆项目
git clone <repository-url>
cd marathon-pb-research

# 安装前端依赖
npm install

# 启动前端开发服务器
npm run dev

# 安装后端依赖
cd server
npm install

# 启动后端服务
npm start
```

### 生产部署

```bash
# 构建前端
npm run build

# 部署后端
cd server
npm install --production
npm start
```

## 🔧 配置说明

### 环境变量

```env
# 后端服务端口
PORT=3002

# 数据更新频率（cron表达式）
UPDATE_CRON=0 0 6 * * *
STATUS_REFRESH_CRON=0 0 18 * * *
```

### 数据文件

- `server/data/races.json` - 赛事数据
- `server/data/feedback.json` - 用户反馈
- `server/logs/` - 运行日志

---

## 📝 API接口

### 赛事相关
- `GET /api/races` - 获取所有赛事
- `GET /api/races/:id` - 获取单个赛事
- `GET /api/races/search?q=` - 搜索赛事
- `POST /api/races/update` - 手动更新数据

### 反馈相关
- `POST /api/feedback` - 提交反馈
- `GET /api/admin/feedback` - 获取所有反馈
- `PUT /api/admin/feedback/:id` - 更新反馈状态
